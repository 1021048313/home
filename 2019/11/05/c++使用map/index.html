    2 说明

    map 是一种容器，用来存储若干，由关键值 (Key Value) 和映射值 (Mapped Value) 配对成的元素。

    key与mapped值存储在成员类型 value_type 中：

    map 中的映射值可以使用括号运算符 (operator[]) 通过其关联的 Key 值直接访问。

    map 通常使用二叉搜索树实现。

    3 容器属性

    • 关联性：
      • 元素的参考地址为key
    • 有序性：
      • 容器中的元素按一定排序方式
    • 映射
      • 一个key->一个mapped
    • 唯一key
